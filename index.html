Laporan Perencanaan Website HTML

Tugas 1 - Pemrograman Web

1. Identitas Website

Nama Website: Laundry Kilap

Tema Website: Modern dan Bersih (Clean & Fresh). Menggunakan palet warna biru muda, putih, dan abu-abu lembut untuk menciptakan kesan rapi, cepat, dan profesional.

2. Isi Website (Content)

Website ini dirancang sebagai landing page satu halaman (Single Page Website) yang informatif dan mudah digunakan untuk menarik pelanggan laundry.

  • Hero Section: Gambar layanan laundry yang bersih, slogan layanan cepat dan rapi, serta tombol ajakan bertindak seperti "Pesan Sekarang".
  • Tentang Kami (About Us): Penjelasan singkat mengenai Laundry Kilap, komitmen terhadap kualitas, kebersihan, dan layanan antar jemput.
  • Layanan Utama: Menampilkan paket laundry seperti cuci kiloan, cuci kering, dan setrika lengkap dengan deskripsi singkat.
  • Kontak & Lokasi (Contact & Location): Informasi jam operasional, alamat lengkap, nomor telepon/WhatsApp, dan formulir pesan cepat.

3. Struktur Menu Website

Karena dirancang minimalis dalam satu halaman, menu navigasi akan menggunakan anchor links untuk memudahkan pengunjung menuju setiap bagian.

[Header / Logo] ─── Beranda (Home) ─── Tentang (About) ─── Layanan (Services) ─── Kontak (Contact)

Setiap menu akan mengarahkan pengguna ke bagian yang relevan tanpa perlu memuat ulang halaman.

4. Fitur yang Akan Dibuat

Untuk memastikan website laundry fungsional dengan HTML dan CSS dasar, berikut fitur-fitur yang akan diimplementasikan:

  1. Navigation Bar Responsif: Bar navigasi tetap terlihat saat halaman digulir untuk memudahkan akses cepat.
  2. Hover Effect pada Daftar Layanan: Kartu layanan menampilkan efek visual saat disorot untuk membuat halaman lebih interaktif.
  3. Formulir Kontak Sederhana: Formulir
    dengan validasi dasar seperti required pada nama, email, dan pesan agar pelanggan dapat mengirim permintaan laundry.
  4. Footer Media Sosial: Tautan ke akun WhatsApp dan Instagram untuk memudahkan pelanggan menghubungi dan memesan layanan.
